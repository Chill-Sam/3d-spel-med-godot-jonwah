# Projektplan

## Bakgrund
Projektets kriterier är att bygga ett 3d-spel som innehåller följande delar:
-	Grundläggande spelmekanik (Rörelse, förlust, vinst)
-	Grafik
-	UI
-	Ljudeffekter och musik
-	AI & spelelement
-	Scen och nivåer

Spelet skall även kunna distribueras via en .exe fil.


## Projektbeskrivning
Spelet ska vara ett 3D high speed platformer first person shooter. Spelet går ut på att navigera platform levlar i första person och attackera fiender med vapen. Det kommer att finnas ett UI med både health och vapen som man har. Fienderna kommer att ha ett AI system som låter dem attackera spelaren. Det kommer finnas melee vapen, skjutvapen och en grappling hook. I slutet av vissa levlar kommer det att finnas bossar som kräver både platforming och skjutande för att döda. Spelets huvudgimmick är att man kan sakta ner tiden i några sekunder. Man kommer att kunna dö, starta och stoppa spelet. Dessutom kommer spelet att ha ljudeffekter och musik för de olika nivåerna och bossar.
